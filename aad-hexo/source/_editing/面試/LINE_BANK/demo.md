


```
問題描述
給定創建矩形所需的4個點中的3個坐標，我們想找到另一個點的坐標。給定一個數組v，其中包含三個點的坐標作為參數，請完成求解函數以返回創建矩形所需的另一點的坐標。但是，矩形的每一邊都平行於x軸和y軸，並且僅在可以製作矩形時才作為輸入給出。

限制
v是包含三個點的坐標的二維數組。
v的每個元素表示點的坐標，並且坐標以[x軸坐標，y軸坐標]的順序給出。
坐標值為1到10億的自然數。
以[x軸坐標，y軸坐標]的順序返回製作矩形所需的另一點的坐標。
I / O示例
v	結果
[[1，4]，[3，4]，[3，10]]	[1，10]
[[1，1]，[2，2]，[1，2]]	[2，1]
I / O示例說明
輸入/輸出示例＃1當
三個點分別位於位置[1，4]，[3，4]和[3，10]上時，如果這些點位於[1，10]，則它成為一個矩形。

輸入/輸出示例＃2當
三個點分別位於位置[1，1]，[2，2]和[1，2]上時，如果這些點位於[2，1]上，則它將變成一個矩形
```

```
function solution(v) {

    var Xarr = [];
    var Yarr = [];
    for(var i=0; i< v.length; i++) {
        Xarr.push(v[i][0])
        Yarr.push(v[i][1])
    }
   var a = removeDelupate(Xarr);
   var b = removeDelupate(Yarr);
    var answer = [a,b];
    console.log(answer)
    return answer;
}

function removeDelupate(arr) {
    var [ a1, a2, a3 ] = arr;
    if(a1 === a2) return a3;
    if(a1 === a3) return a2;
    if(a2 === a3) return a1;

}
```
